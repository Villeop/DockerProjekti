Docker Docker Compose User Notes

Kuvaus:

Tavoitteena oli luoda Docker pohjainen ympäristö, jolla voitaisiin esitellä kurssilla käytyjä aiheita nettisivujen muodossa. Tässä projektissa nettisivut ovat jaettu seuraavasti:

  • Docker: Dockerin peruskäsitteet ja toiminta.
  • Docker Compose: Docker Composea ja sen käyttö.
  • User Notes: Käyttäjän omat muistiinpanot.

Tämä sivusto on toteutettu Flaskilla ja Bootstrapilla.

Voit siirtyä sivulta toiselle yläpalkin linkeistä.
